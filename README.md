# App de Clima Minimalista

Proyecto web sencillo en JavaScript para obtener datos meteorológicos usando la API de Open-Meteo.

## Estructura del proyecto

- `index.html` - Página principal con el formulario de búsqueda y la tarjeta de resultados.
- `css/styles.css` - Estilos oscuros y minimalistas para la interfaz.
- `js/script.js` - Lógica para buscar la ubicación, consultar Open-Meteo y mostrar el clima.

## Cómo usar

1. Abrir `index.html` en el navegador.
2. Escribir el nombre de la ciudad.
3. Presionar `Buscar` para ver el clima actual.

## Características

- Tema oscuro y diseño minimalista.
- Búsqueda de ciudad con geocodificación de Open-Meteo.
- Muestra temperatura, viento, dirección y hora del reporte.
